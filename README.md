# RoomIR_audio_conv
Room impulse response and audio signal convolution. 

## Description
This code convolves a room impulse response and an audio signal. The output is an audio with modified reverberant characteristics. It also plots the original and the modified signals in frequency domain.

Song used as an exemple: \
Living Nightmare by snowflake\
Ft: Blue Wave Theory\
